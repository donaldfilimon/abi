//! Web Feature Overview
//!
//! Provides HTTP client utilities and higher‑level web integrations. The
//! directory contains:
//!   - `mod.zig` – Public aggregation.
//!   - `client.zig` – Async HTTP client implementation.
//!   - `weather.zig` – Example service that demonstrates fetching external
//!     data.
//!
//! New web‑based services can be added by creating a new `.zig` file and
//! exposing its API via `mod.zig`.
