# ABI VS Code Extension (scaffold)

This folder contains a minimal VS Code extension scaffold for ABI development. It registers a single command `ABI: Hello World`.

How to run
1. Install dependencies: `npm install`
2. Compile: `npm run compile`
3. Press F5 in VS Code to run the Extension Host with the extension loaded.

Notes
- This is a scaffold. Integrate the Zig LSP or other features in follow-up tasks.
