Observability covers logging, metrics, profiling, and health checks. These tools help validate performance and reliability during deployment.

Logging

Logs use structured formats and include module names. Set log levels per environment to balance signal and noise.

Metrics and profiling

zig build -Denable-profiling=true
zig build run -- system-info

Health checks

Streaming services expose health endpoints. Integrate them with load balancers and container orchestration for automatic restarts.
