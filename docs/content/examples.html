Examples are compiled and wired into the build system. Use them to validate your toolchain and learn module usage patterns.

zig build examples

zig build run-hello

zig build run-agent

Where to look

Example sources live in the examples/ directory and are kept small and focused. They are a good starting point for custom integrations.
