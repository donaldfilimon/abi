This guide walks through a minimal setup and a first run of the ABI CLI. The project is written for Zig 0.16 and expects a standard toolchain install.

Prerequisites

  • Zig 0.16 in your PATH
  • Git and a Unix style shell for running scripts
  • Optional: a GPU backend if you plan to test acceleration

Build and run

zig build
zig build run -- --help

Run an example

zig build run-hello
zig build run-database

Enable or disable features

Feature flags control major modules. Use build options to enable or disable modules for your build: 

zig build -Denable-ai=true -Denable-gpu=false
zig build -Denable-database=true -Denable-network=false

Next steps

Review the Configuration page for runtime tuning and the CLI page for commands. Module pages explain AI, GPU, database, and network capabilities.
