The database module implements WDBX, a vector database optimized for similarity search and embedding workflows. It supports multiple indexing strategies and batch ingestion.

Core operations

  • Insert vectors with metadata
  • Similarity search with configurable distance metrics
  • Index rebuilds and optimization commands

CLI examples

zig build run -- db add --file vectors.jsonl
zig build run -- db query --text "hello world"
zig build run -- db stats

Backups

Use the CLI to create and restore snapshots before large migrations. Keep database paths on fast local storage when possible.
