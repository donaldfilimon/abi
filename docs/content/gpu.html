The GPU module provides a unified backend interface for acceleration. Backends can be selected at build time and discovered at runtime.

Backend selection

zig build -Dgpu-backend=vulkan
zig build -Dgpu-backend=cuda,vulkan
zig build -Dgpu-backend=none

Runtime behavior

  • Automatic fallback to CPU when a backend is unavailable
  • Explicit backend selection via ABI_GPU_BACKEND
  • Memory pools tuned for large model workloads

GPU health checks

zig build run -- gpu status
zig build run -- gpu devices
