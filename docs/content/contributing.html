Contributions are welcome. Please follow the repo conventions and keep changes focused. Use small, reviewable commits and describe the intent in the message.

Local workflow

zig fmt .
zig build test --summary all

Code style

  • 4 spaces for indentation, no tabs
  • Prefer explicit imports over usingnamespace
  • Use {t} for error and enum formatting

Commit messages

Use Conventional Commits with a clear scope when applicable: 

feat(gpu): add pooled allocator
docs: refresh configuration guide
