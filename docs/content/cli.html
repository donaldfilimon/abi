The CLI is the primary entry point for running services and utilities. Build it with Zig, then run commands through the build system or the installed binary.

Run the CLI

zig build run -- --help
zig build run -- system-info

Common commands

zig build run -- db stats
zig build run -- llm chat --model llama-7b
zig build run -- gpu status

Global patterns

  • Use zig build run -- to pass CLI arguments
  • Set environment variables to override defaults
  • Prefer feature flags at build time to trim the binary

Examples as commands

Examples are wired into the build system. Run them without manual compilation: 

zig build run-hello
zig build run-gpu
