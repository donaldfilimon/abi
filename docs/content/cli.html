The CLI is the primary entry point for running services and utilities. Build it with Zig, then run commands through the build system or the installed binary.

Run the CLI

zig build run -- --help
zig build run -- system-info

Available commands

CommandPurpose
dbDatabase operations (add, query, stats, optimize, backup, restore, serve)
agentAI agent interaction (interactive, one-shot, 13 personas)
llmLLM inference (chat, generate, serve, info, bench, download, list)

gpu-dashboardInteractive GPU + Agent monitoring TUI
benchBenchmarks (all, simd, memory, ai, quick, concurrency)

Model management

zig build run -- model list                           # List cached models
zig build run -- model info llama-7b                  # Show model details
zig build run -- model download TheBloke/Model:Q4_K_M # Download from HuggingFace
zig build run -- model remove llama-7b                # Remove cached model
zig build run -- model search llama                   # Search HuggingFace models
zig build run -- model path llama-7b                  # Get local model path

Common commands

zig build run -- db stats
zig build run -- llm chat --model llama-7b
zig build run -- gpu status

Global patterns

  • Use zig build run -- to pass CLI arguments
  • Set environment variables to override defaults
  • Prefer feature flags at build time to trim the binary

Examples as commands

Examples are wired into the build system. Run them without manual compilation: 

zig build run-hello
zig build run-gpu
