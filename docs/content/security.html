Security defaults are conservative but configurable. Production deployments should enable authentication, secrets management, and TLS.

Secrets

  • Set ABI_MASTER_KEY to protect sensitive configuration
  • Store API keys in environment variables or secret stores
  • Rotate tokens on a regular schedule

Authentication

Streaming endpoints can use bearer token auth. Enforce a strong token policy and limit access to trusted networks.

Safe defaults

  • Disable unused features at build time
  • Use read-only tokens for clients
  • Enable rate limiting for public services
