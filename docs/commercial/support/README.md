# Support Services

## Support Channels

### Community (Free)
- GitHub Issues
- GitHub Discussions
- Community Discord
- Stack Overflow tag: `abi-framework`

### Paid Support
- Private Slack channel
- Email support
- Video calls (Premium+)
- Phone support (Enterprise)

## Support Scope

### In Scope
- Installation and configuration
- API usage questions
- Bug investigation
- Performance troubleshooting
- Version upgrades
- Security advisories

### Out of Scope
- Custom feature development (see [Custom Dev](../enterprise/custom-dev.md))
- Third-party integrations not in docs
- Non-ABI code debugging
- Training (see [Education](../../education/))

## Response Times

| Priority | Standard | Premium | Enterprise |
|----------|----------|---------|------------|
| Critical | 48 hours | 8 hours | 1 hour |
| High | 72 hours | 24 hours | 4 hours |
| Normal | 5 days | 48 hours | 24 hours |
| Low | Best effort | 5 days | 48 hours |

## Priority Definitions

- **Critical:** Production down, no workaround
- **High:** Major feature broken, workaround available
- **Normal:** Minor issue, standard questions
- **Low:** Feature requests, nice-to-haves

## Escalation Path

1. Support engineer
2. Senior engineer
3. Core maintainer
4. Project lead

## Hours of Operation

- **Standard/Premium:** Monday-Friday, 9am-6pm (Your timezone)
- **Enterprise:** 24/7/365
