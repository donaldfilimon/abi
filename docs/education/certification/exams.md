# Certification Exams

## Exam Format

| Level | Questions | Time | Passing Score |
|-------|-----------|------|---------------|
| Developer | 30 | 45 min | 70% |
| Professional | 50 | 90 min | 80% |
| Expert | 40 | 120 min | 90% |

## Question Types

- Multiple choice
- Code completion
- Error identification
- Architecture design

## Exam Topics

### Developer Exam
- Installation and setup
- Basic API usage
- Configuration
- Error handling
- Testing basics

### Professional Exam
- GPU acceleration
- AI integration
- Database operations
- Performance basics
- Project structure

### Expert Exam
- Custom implementations
- Advanced optimization
- Distributed systems
- Production deployment
- Architecture design

## Scheduling

1. Complete the prerequisite course
2. Request exam via GitHub issue
3. Receive exam link (valid 7 days)
4. Complete within time limit
5. Receive results within 24 hours

## Retake Policy

- Wait 7 days between attempts
- No limit on retakes
- Highest score counts
