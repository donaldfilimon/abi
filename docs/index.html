Overview

The ABI framework provides a modular, high-performance platform for building AI/ML applications. Key features include:

  • AI & ML: Neural networks, embeddings, and machine learning algorithms
  • Vector Database: High-performance vector storage and search
  • GPU Acceleration: CUDA and Vulkan compute backends
  • Web Services: REST APIs and WebSocket support
  • Monitoring: Observability and performance profiling
  • Plugin System: Extensible architecture with plugin support

Quick Start

const abi = @import("abi");

// Initialize framework
var framework = try abi.init(allocator, .{});
defer abi.shutdown(&framework);

// Use AI features
const ai = abi.ai;
// ... build and train models
