# AI Plan Superpower Plugin

Core AI planning capabilities for OpenCode within the ABI framework.

## Capabilities

- AI subsystem integration
- Plugin framework registration
- Runtime lifecycle management
- Configuration and settings management
- Status monitoring and reporting

## Integration Points

- ABI's AI subsystem integration
- OpenCode plugin framework integration
- Runtime lifecycle management
- Configuration and settings management

## Responsibilities

- Model selection and preparation for agent tasks
- Training data management and pipeline configuration
- Inference planning and optimization
- Integration between AI models and agent workflows
- Performance monitoring and resource allocation

## Implementation

Works with ABI's AI subsystem (`src/features/ai/`), model catalog, router, and training infrastructure (`src/features/ai/training*.zig`).

See also the top-level `ai-plan` skill and `abi-superpower-ai`.
