# MCP Superpower Plugin

Model Context Protocol integration for connecting with external systems, tools, and services.

## Capabilities

- JSON-RPC 2.0 communication protocol
- Tool listing and execution protocols
- Resource and prompt management
- Transport layer (stdio, HTTP/SSE)
- Authentication and authorization

## Integration

Connects with ABI's MCP server, handlers, and connector systems to provide external tool integration and model communication.
