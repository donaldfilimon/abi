Install

Quick Start

From zero to running in 4 commands.

1

Clone

git clone https://github.com/donaldfilimon/abi.git
2

Build with features

zig build -Denable-ai=true -Denable-gpu=true
3

Run tests

zig build test --summary all
4

Launch

zig build run -- tui
CLI

Command Line

$zig build run -- tuiInteractive UI
$zig build run -- agentAI Agent
$zig build run -- db statsDB Stats
$zig build run -- gpu backendsGPU Info
$zig build run -- llm chat model.ggufChat
$zig build run -- train run --epochs 10Train
$zig build run -- --list-featuresFeatures
$zig build run -- system-infoSystem
